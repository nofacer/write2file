# how to install 
`pip install write2file`
# how to use
```python
from write2file import write2file
```
## Create new file
```python
write2file(output_path,['header1', 'header2', '...', 'headern'], True)
```
## Write into existing file
```python
write2file(output_path, [content1,content2,...,contentn])
```
